HiRemoteMenuBar를 시뮬레이션하는 컨트롤바 창을 생성합니다.

2단계: 메인 창 최소화

이 창(ezHelp 메인 창)을 최소화합니다.

3단계: RemoteManager에서 포커스 테스트

1. RemoteManager를 실행하고 프로세스 감지
2. 이 Chrome 프로세스를 찾기 (ezHelp로 표시됨)
3. 🎯 포커스 버튼 클릭
4. 메인 창과 컨트롤바가 모두 복원되는지 확인
